# Ray2Fix
Ray2Fix is a collection of fixes and a configuration tool that aims to simplify setting up Rayman 2 for the average user.

It can also be used as a basic mod loader (can be enabled in Tweaks tab in R2FixCfg.exe).

## How to Install
### Prerequisites:
- The [GOG or the delisted Steam versions](https://www.gog.com/game/rayman_2_the_great_escape) of the game.
- Uplay/Ubisoft Connect and Retail CD/DVD versions are not supported.
### Guide:
- Download the **[latest release](https://github.com/icup321/Ray2Fix/releases/latest)** and extract all files to Rayman 2's installation directory.
- Run R2FixCfg.exe and choose your preferred display settings.
- Play the game.
